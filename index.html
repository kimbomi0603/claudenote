🗂️ 최근 회의

보고서를 만들면 자동 저장돼요. 누르면 다시 열기 · 꾹 누르면 바로 공유 · ✕로 삭제.

🎙️ 내 음성 파일

추가한 파일은 앱 안에 보관되어 새로고침해도 그대로 남아요. 항목을 누르면 재생 · ⬇ MP3 저장 · 📝 받아쓰기 · ✕ 삭제.
💡 PLAUD처럼 녹음 파일을 넣으면 자동으로 받아쓰기가 시작돼요(기기 안 무료 처리, 서버·비용 없음, 첫 실행 때 엔진 다운로드). 여러 개를 넣었다면 각 파일의 📝 버튼 한 번이면 됩니다. 구간별 타임스탬프가 붙어 전사에서 해당 위치를 바로 재생할 수 있어요. 긴 회의는 처리가 느릴 수 있으니, 그럴 땐 갤럭시 음성 녹음 앱의 ‘텍스트 변환’ 결과를 붙여넣거나 공유 → 클로드노트로 보내면 더 빨라요.

📝 전사 입력

갤럭시 ‘텍스트 변환’ 결과나 PLAUD 전사를 붙여넣으면, 회의록·전체 전사·할 일이 자동으로 만들어지고 바로 공유할 수 있어요.
🎙️ 조용한 녹음기 모드: 녹음 중엔 화면이 저절로 꺼지지 않게 잠금(Wake Lock)되고 상단에 경과 시간이 표시돼요. 음성은 파일로 저장되어 목록에 들어가고(⬇ 버튼으로 MP3 저장 가능), 문장마다 [분:초] 타임스탬프가 붙으며, 조용해져도 자동으로 이어 녹음됩니다. 다른 앱으로 잠깐 전환해도 녹음은 유지돼요(받아쓰기는 돌아오면 재개 — 녹음 파일은 나중에 📝로 전체 받아쓰기 가능). ※ 통화 녹음과 화면을 완전히 끈 장시간 녹음은 웹앱 보안 정책상 불가 — 그 경우 폰 기본 녹음앱으로 녹음한 뒤 파일을 여기 불러오면 자동 전사됩니다.

🗣️ 발화자 설정

전사에서 감지된 화자입니다. 이름·역할·색을 정하면 모든 보고서에 반영돼요. (목소리 자동 식별이 아니라, 화자에 이름을 지정하는 방식입니다.)
먼저 ‘시작’ 탭에서 전사를 입력하고 보고서 자동 생성을 눌러주세요.
전체 전사에서 발언마다 화자를 바꿀 수도 있어요(보고서 탭 → 전체 전사). PLAUD가 한 사람을 둘로 나눴다면, 같은 이름·색을 지정하면 합쳐 보입니다.
전사를 입력하면 세 가지 보고서가 자동 생성됩니다.
